# Population Information Search

## Tools Used:

1. Laravel
2. Database: MySQL
3. Inertia.js
4. Vue

## To run project:

1. composer install
2. npm insall
3. php artisan migrate:fresh --seed OR you can import population_db.sql(contains dummy data) file located in the root folder
4. php artisan serve
5. in new console run: npm run dev
